## Project Vision
RIDOS-Core is a next-generation Linux distribution built on:
- Linux Kernel 6.12 LTS with Rust support enabled (CONFIG_RUST=y)
- GNOME Wayland as the primary desktop
- Calamares installer with LUKS full-disk encryption
- Optional tool system — base OS is lean, extras are your choice
- Roadmap: Linux 7.0 migration, Rust coreutils, Panic Key v2.0
